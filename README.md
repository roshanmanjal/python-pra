## What's Inside? 📚

**Chapter 01 - First Steps**  
Just getting started with Python basics - printing poems, playing with the OS module, and even making my computer talk!

**Chapter 02 - The Basics**  
Variables, data types, operators... you know, the foundation stuff. Plus some practice problems to actually use what I learned.

**Chapter 03 - Playing with Strings**  
Learned how to slice, dice, and manipulate strings. Super useful for text processing!

**Chapter 04 - Lists & Tuples**  
Working with collections of data. Lists are awesome because you can change them, tuples not so much!

**Chapter 05 - Dicts & Sets**  
Dictionaries are like real dictionaries (word -> meaning), and sets are for unique stuff. Pretty cool!

**Chapter 06 - Making Decisions**  
If this, then that. Simple but powerful! Built some fun things like a spam detector and grade calculator.

**Chapter 07 - Loops All Day**  
For loops, while loops, break, continue... Making the computer do repetitive stuff so I don't have to!

**Chapter 08 - Functions**  
Finally learned to write reusable code! Recursion was mind-bending at first but makes sense now.

**Chapter 09 - Files**  
Reading and writing files. Created a word search program that reads from a text file.

**Chapter 10 - Classes & Objects**  
My introduction to OOP. Made a student class, employee class, calculator class, and even a train booking system!

**Chapter 11 - Inheritance**  
Getting fancy with OOP - classes inheriting from other classes. Also learned about decorators and the super() method.

**Chapter 12 - Advanced Stuff**  
Exception handling (so my programs don't crash), list comprehensions (one-liners ftw!), match-case, and the walrus operator.

**Chapter 13 - Pro Level**  
Virtual environments, lambda functions, map/filter/reduce - the tools that make Python really powerful.
